# Črnilo Sever

Kompletna statična spletna stran za tattoo studio.

## Kaj je vključeno

- index.html
- assets/css/style.css
- assets/js/main.js
- assets/img mapa za poznejše fotografije
- netlify.toml za hitro objavo na Netlify
- README z navodili

## Kako odpreš stran

1. Razširi ZIP.
2. Odpri datoteko index.html v brskalniku.
3. Za objavo na Netlify povleci celotno mapo v Netlify Drop.

## Obrazec

Obrazec trenutno deluje v demo načinu.
Podatki se izpišejo v konzolo brskalnika.

Za pravo pošiljanje:
1. Odpri assets/js/main.js.
2. Poišči vrstico VPIŠI_SVOJ_ENDPOINT.
3. Vpiši povezavo do Google Apps Script, Formspree, Netlify Forms ali Make webhook.
4. Odkomentiraj fetch blok.

## Fotografije

V mapi assets/img lahko dodaš fotografije tetovaž.
Trenutna galerija uporablja CSS bloke, zato stran deluje tudi brez slik.
